Selecionando Elementos

Métodos comuns para selecionar elementos do DOM:


// Selecionar um elemento
document.getElementById('meuId');

// Selecionar vários elementos
document.querySelectorAll('.minhaClasse');

// Selecionar o primeiro elemento que corresponde
document.querySelector('div');

Modificando Elementos

Alterando conteúdo e estilos de elementos:


// Alterar conteúdo de texto
elemento.textContent = 'Novo texto';

// Alterar HTML interno
elemento.innerHTML = '<strong>Texto em negrito</strong>';

// Alterar estilos
elemento.style.color = 'red';
elemento.style.fontSize = '20px';

// Adicionar/remover classes
elemento.classList.add('nova-classe');
elemento.classList.remove('classe-antiga');

Event Listeners

Como adicionar interatividade com eventos:


// Adicionar um event listener
botao.addEventListener('click', function() {
    console.log('Botão clicado!');
});

// Evento de formulário
formulario.addEventListener('submit', function(event) {
    event.preventDefault(); // Evita o recarregamento
    console.log('Formulário enviado');
});
