Variáveis

Variáveis são containers para armazenar valores de dados. Em JavaScript temos 3 formas de declarar variáveis:


// var - escopo de função (mais antigo)
var nome = "João";

// let - escopo de bloco (pode ser reatribuída)
let idade = 30;

// const - escopo de bloco (não pode ser reatribuída)
const PI = 3.14;
Dica: Prefira usar const por padrão e let quando precisar reatribuir valores. Evite var.

Tipos de Dados

JavaScript tem tipos de dados dinâmicos, o mesmo valor pode conter diferentes tipos:

Tipo Descrição Exemplo
String Texto "Olá Mundo"
Number Números inteiros ou decimais 42 ou 3.14
Boolean Verdadeiro ou falso true ou false
Array Lista ordenada [1, 2, 3]
Object Coleção de pares chave-valor {nome: "João", idade: 30}
