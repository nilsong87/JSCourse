Objetos Literais

A forma mais simples de criar objetos em JavaScript:


const pessoa = {
    nome: 'João',
    idade: 30,
    saudar() {
        return `Olá, meu nome é ${this.nome}`;
    }
};

console.log(pessoa.saudar()); // "Olá, meu nome é João"

Factory Functions

Funções que retornam objetos (padrão antes das classes):


function criarPessoa(nome, idade) {
    return {
        nome,
        idade,
        saudar() {
            return `Olá, eu sou ${this.nome} e tenho ${this.idade} anos`;
        }
    };
}

const maria = criarPessoa('Maria', 25);
console.log(maria.saudar());

Protótipos

JavaScript é baseado em protótipos, não em classes (até ES6):


function Pessoa(nome, idade) {
    this.nome = nome;
    this.idade = idade;
}

Pessoa.prototype.saudar = function() {
    return `Olá, meu nome é ${this.nome}`;
};

const joao = new Pessoa('João', 30);
console.log(joao.saudar());

Demonstração de Classes

Herança com Classes

ES6 introduziu a sintaxe de classes, que é "syntactic sugar" sobre o sistema de protótipos: 


    class Animal {
    constructor(nome) {
    this.nome = nome;
    }
    
    
    falar() {
        return `${this.nome} faz um som.`;
    }
    }
    
    class Cachorro extends Animal {
    falar() {
    return ${this.nome} late!;
    }
    }
    
    const rex = new Cachorro('Rex');
    console.log(rex.falar()); // "Rex late!"
